# 🏗️ SpecLogic Universal Engine
**Automated Compliance Auditing for 2026 Residential Construction Standards**

## 💡 Overview
This tool was built to solve a specific $2,500+ problem in the San Diego construction market: **Title 24 Compliance Gaps.** As of March 2026, local amendments have made manual submittal reviews prone to expensive errors.

## 🚀 Features
- **Dynamic Market Switching:** Uses a `config.json` architecture to pivot between different municipal codes instantly.
- **AI-Powered OCR:** Utilizes `PyMuPDF` and `FuzzyWuzzy` to scan technical blueprints and specs for compliance keywords.
- **Automated Reporting:** Generates a professional audit trail for Project Managers.

## 🛠️ Stack
- **Frontend:** Streamlit (Python-based SaaS UI)
- **Engine:** Python 3.12, PyMuPDF, Pandas
- **Deployment:** Streamlit Community Cloud
